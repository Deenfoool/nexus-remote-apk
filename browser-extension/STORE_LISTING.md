# Store Listing

## Name

Nexus Remote Browser Bridge

## Short description

Connect YouTube and VK media tabs to Nexus Remote PC for remote playback control from your phone.

## Full description

Nexus Remote Browser Bridge links supported browser media tabs with Nexus Remote PC on the same computer.

It helps Nexus Remote detect what is currently playing in your browser and enables better playback controls from your Android phone.

Features:

- works with YouTube, YouTube Music, and VK;
- reads media metadata such as title, artist, playback state, progress, and artwork;
- sends media data only to the local Nexus Remote PC app on your computer;
- supports browser playback commands such as play/pause, seek, next, previous, mute, subtitles, fullscreen, and volume actions when available.

The extension is intended to work together with:

- `Nexus Remote PC` on Windows;
- `Nexus Remote` on Android.

No cloud account is required. Communication stays local.
