# Установка Nexus Remote на Android

## Что установить

Для прямой установки используйте `app-release.apk`.

Путь после локальной сборки:

```text
C:\Users\salum\AndroidStudioProjects\nexusremote\app\build\outputs\apk\release\app-release.apk
```

`app-release.aab` нужен для публикации в магазине.

## Установка APK

1. Откройте `app-release.apk` на телефоне.
2. Разрешите установку из неизвестного источника, если Android попросит.
3. После установки откройте `Nexus Remote`.
4. На первом запуске дайте доступ к камере и уведомлениям.

## Первое подключение

1. Убедитесь, что `Nexus Remote PC` уже запущен на компьютере.
2. На телефоне нажмите `Сканировать QR сопряжения`.
3. Наведите камеру на QR из окна PC-приложения.
4. Подтвердите подключение на ПК.
