# Nexus Remote

Android-приложение для управления ПК в локальной сети. Текущий прототип умеет:

- двигать курсор с телефона как с тачпада;
- отправлять левый и правый клик;
- управлять медиа: play/pause, prev/next, громкость, mute;
- отправлять Enter, Esc и Win+D.

## Запуск ПК-сервера

На Windows можно просто открыть:

```text
pc-server\start_nexus_remote_server.bat
```

Или запустить из PowerShell:

```powershell
python .\pc-server\nexus_remote_server.py --token nexus
```

Сервер покажет подходящие IP, порт и ключ. Эти значения нужно ввести в Android-приложении.

Если телефон не подключается:

- проверьте, что телефон и ПК в одной Wi-Fi/LAN сети;
- разрешите Python входящие подключения в Windows Firewall;
- используйте IP из вывода сервера, а не `127.0.0.1`.

## Протокол

Приложение отправляет `POST http://<pc-ip>:8765/command`:

```json
{
  "token": "nexus",
  "type": "mouse_move",
  "payload": { "dx": 12, "dy": -4 }
}
```

Ключ нужен, чтобы случайное устройство в сети не могло отправлять команды без минимальной авторизации.
